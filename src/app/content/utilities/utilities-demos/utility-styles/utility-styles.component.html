CSS Utility Styles & Classes

Covalent includes many utility styles to save you time. We've included general utilities, padding (pad), margin (push), negative margin (pull), sizing, and text classes.

General Utilities

.radius-none // no border radius .overflow-hidden // hide overflow .overflow-auto // normal overflow .block // display block .inline-block // display inline block .relative // relative positioning .fixed // fixed positioning .height-auto // 0 min height .z-3 // z index 3 .z-2 // z index 2 .z-1 // z index 1
