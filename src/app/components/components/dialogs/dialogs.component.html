TdDialogService

Service provided with methods that wrap the material2 [MdDialog] service and provide an easier experience for simple dialogs.

Note: if no [ViewContainerRef] is provided, [TdDialogService] will throw an error.

Methods:

The [TdDialogService] service has {{dialogServiceMethods.length}} properties:

Example(after setup):

Typescript:

{ if (accept) { // DO SOMETHING } else { // DO SOMETHING ELSE } }); } openPrompt(): void { this._dialogService.openPrompt({ message: 'This is how simple it is to create a prompt with this wrapper service. Prompt something.', viewContainerRef: this._viewContainerRef, //OPTIONAL title: 'Prompt', //OPTIONAL, hides if not provided value: 'Prepopulated value', //OPTIONAL cancelButton: 'Cancel', //OPTIONAL, defaults to 'CANCEL' acceptButton: 'Ok', //OPTIONAL, defaults to 'ACCEPT' }).afterClosed().subscribe((newValue: string) => { if (newValue) { // DO SOMETHING } else { // DO SOMETHING ELSE } }); } } ]]>

You can also set a default [ViewContainerRef] and avoid having to use it on each dialog open method call.
