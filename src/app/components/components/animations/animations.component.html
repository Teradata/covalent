Animations Core Covalent animation utilities

Utilities to help add simple animations. For custom and complex animations look into the Angular animations docs.

Setup:

Import individual animation utilities needed into the component that will contain the elements you want to animate (ex: TdRotateAnimation).

Invoke the utility functions with a required anchor name and optional configs in the component's metadata. The anchor name is used to attach the animation to an arbitrary element in the template.

 Animations Docs Rotate Animation

Use TdRotateAnimation to rotate any element based on a boolean state.

Method Options:

TdRotateAnimation(anchor: string, duration: number, degrees: number)

anchor

The anchor name is used to attach the animation to an arbitrary element in the template.

duration

Duration the animation will run in miliseconds. Defaults to 250 ms.

degrees

Degrees of rotation that the dom object will animation. Defaults to 180 degrees.
