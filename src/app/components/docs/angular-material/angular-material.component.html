Angular Material Material Design components for Angular 2 apps

Core Components

By default, the UI Platform includes all the core Angular-Material components. Obviously Angular 2.0 and therefore Angular-Material are brand new, so we'll be constantly updating to pull them down for you and we'll also be adding additional missing components. The current list is:

  • button-toggle
  • button
  • card
  • checkbox
  • grid-list
  • icon
  • input
  • list
  • progress-bar
  • progress-circle
  • radio
  • sidenav
  • slide-toggle
  • tabs
  • toolbar

New & Custom Components

With the UI Platform and your app usage, we suggest to follow the official material spec as closely as possible for these reasons:

  • So our products remain consistent
  • To be able to use public material design resources
  • To onboard new hires more easily
  • So devs can easily work on multiple products
  • To enable easy updates and pull down the latest

Obviously we'll need to extend and customize material, but when doing so try to submit to the main UI Platform repo and collaborate with others to ensure standards.
