Creating new views Harness the power of the new Angular CLI

Generate Route

Using the CLI, nagivate into the directory you want the new route, then simply:

ng generate route my-route

The CLI will do the heavy lifting and:

  • Generate the directory, css, html, spec.ts, component.ts & index.ts
  • Add an @import and new route in the parent directory
  • Update system-config.ts for you!

Basically it does all the irritating little things for you.

Now you simply creating your route html and logic for this new "view"!

Make sure you import any directives you want, for example in my-route.component.ts:

import { MD_CARD_DIRECTIVES } from '@angular2-material/card'; import { TdHighlightComponent } from '@covalent/highlight';

And define those directives in @Component:

directives: [ MD_CARD_DIRECTIVES, TdHighlightComponent ] Angular CLI Docs
