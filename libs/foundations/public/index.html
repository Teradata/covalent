Code Styling

Code can be displayed inline or as blocks, both using Roboto Mono.

Inline Code

Use inline code for short snippets: const greeting = 'Hello, World!'; or variable names like userName within text.

Code Blocks

// TypeScript example with types
interface User {
  id: number;
  name: string;
  email: string;
}

function greetUser(user: User): string {
  return `Hello, ${user.name}!`;
}

const currentUser: User = {
  id: 1,
  name: 'Jane Doe',
  email: 'jane@example.com'
};

console.log(greetUser(currentUser));

Getting Started

Choose your preferred mode:

Option 1: Utility Classes (What you're seeing now)

Use explicit classes for full control:

<!-- Include theme + utility classes -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@covalent/foundations@latest/themes/light.css">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@covalent/foundations@latest/covalent.css">

<h1 class="cv-headline-1">Title</h1>
<p class="cv-body-1">Paragraph</p>

Option 2: Classless/Semantic (Zero Config)

HTML elements automatically styled - no classes needed! 

<!-- Include theme + classless styles -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@covalent/foundations@latest/themes/light.css">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@covalent/foundations@latest/elements.css">

<h1>Title</h1>  <!-- No classes! -->
<p>Paragraph</p>

npm Install

npm install @covalent/foundations @covalent/tokens

Design Tokens

All styles use CSS variables generated from Covalent design tokens.

Available CSS Variables

  • --cv-theme-primary - Primary brand color
  • --cv-theme-secondary - Secondary accent color
  • --cv-theme-on-surface - Main text color
  • --cv-theme-on-surface-variant - Secondary text
  • --cv-theme-outline - Border color
  • --cv-typography-* - All typography tokens
