Fullscreen directive

Use tdFullScreen directive on any element you want full screen.

Assign tdFullScreen to a template reference variable #myFullScreenElement="tdFullScreen" .

Use the template reference variable to access properties and metheds of the directive:

  • myFullScreenElement.fullScreenIsActive , Check if fullscreen is activated for this element
  • myFullScreenElement.toggleFullScreen() , Toggle fullscreen for this element
  • myFullScreenElement.enterFullScreen() , Active fullscreen for this element
  • myFullScreenElement.exitFullScreen() , Exit fullscreen for this element
